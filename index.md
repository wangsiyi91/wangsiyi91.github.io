- **Estimating Transferable Utility Matching Games with a Nested Fixed Point Approach Using Julia** (2022)

- **Estimating BLP Demand Using Julia** (2022)
	- Estimated demand for differentiated products using MPEC, simulated markets, and performed merger counterfactual analysis.
- **Calibrating Real Business Cycle Model with Post-War U.S. Economy Statistics Using MATLAB** (2020)
	- Built and calibrated a real business cycle model using dynamic programming and matched model statistics to U.S. data.
- **Estimating the Performance of Tele-Secondary School and Normal School in Mexico with Generalized Roy Model Using R** (2020)
	- Compared tele-secondary and regular schooling effectiveness using Roy model, improved classification with new interaction terms.
- **Using Multiple Machine Learning Methods to Estimate Intergeneration Income Mobility of U.S. Census Tracts with “Opportunity Atlas” Database Using R** (2019-2020)
	- Built and compared classification models to identify census tracts promoting income mobility, predicted mobility by race.
